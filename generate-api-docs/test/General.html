Defines the environment in for the output bundle

Property Descriptions
  • includeNodeModules: Whether to include all/none packages (true / false), an array of package names to include, or an object (of a package is not specified, it's included).

  • isLibrary: Whether this is a library build (e.g. less loaders)

  • minify: Whether the output should be minified.

  • scopeHoist: Whether scope hoisting is enabled.

Method Descriptions
  • isBrowser: Whether context specifies a browser context.

  • isNode: Whether context specifies a node context.

  • isElectron: Whether context specifies an electron context.

  • isWorker: Whether context specifies a worker context.

  • isIsolated: Whether context specifies an isolated context (can't access other loaded ancestor bundles).
